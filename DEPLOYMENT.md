## 🔧 Maintenance

### To Update Content
1. Edit `index.html` locally
2. `git add index.html`
3. `git commit -m "Your message"`
4. `git push origin main`
5. Wait 1-2 minutes for GitHub Pages to rebuild

### To Update Images
1. Replace/add images in `images/` folder
2. Update image paths in `index.html` if needed
3. `git add images/`
4. `git commit -m "Update images"`
5. `git push origin main`

### To Update Video
1. Replace video in `videos/output.mp4`
2. `git add videos/output.mp4`
3. `git commit -m "Update video"`
4. `git push origin main`

## 🎯 Custom Domain (Optional)

To use a custom domain like `mindmountain.in`:

1. **Purchase domain** (GoDaddy, Namecheap, etc.)
2. **Configure DNS:**
   ```
   CNAME: www.mindmountain.in → Rahul-Ai07.github.io
   A Record: 185.199.108.153
   A Record: 185.199.109.153
   A Record: 185.199.110.153
   A Record: 185.199.111.153
   ```
3. **Update GitHub Settings:**
   - Settings → Pages → Custom Domain
   - Enter: `mindmountain.in`
   - Enable HTTPS (automatic)
